# AutomationSauceDemo
AutomationSauceDemo là project test tự động cho website Sauce Demo  sử dụng Java, Selenium, TestNG và Page Object Model (POM). Project kiểm tra đăng nhập, thêm sản phẩm vào giỏ hàng, quy trình thanh toán với các scenario hợp lệ và không hợp lệ, đồng thời phát hiện bug như cho phép checkout khi giỏ hàng rỗng, kèm logging và báo cáo kết quả test.
